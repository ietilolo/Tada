The Code for Tada

            
              // Get Values
              function getValues(){
                let start = parseInt(document.querySelector("#startValue").value);
                let end = parseInt(document.querySelector("#endValue").value);
                let numbers = generateNumbers(start, end);
                displayResults(numbers, end);
              }

              // Generate Numbers
              function generateNumbers(start = 1, end = 100){
                let numbers = [];
                for (let i = start; i <= end; i++){
                  numbers.push(i);
                }
                return numbers;
              }

              // Filter list
              function filterValues(numbers, end){
                let filtered = "";
                let templateRow = "";
                let template = [];
                for (let i = 0; i <= numbers.length; i++) {
                  if ( i % 5 == 0 & i != 0 || i == end){
                    templateRow += filtered;
                    template.push(templateRow);
                    filtered = "";
                    templateRow = "";
                  }
                  let td = "<td>";
                  let number = numbers[i];
                  if (number % 3 == 0 && number % 5 == 0) {
                    td += "<span class='red'>Tada</span></td>";
                  } else if (number % 3 == 0){
                    td += "<span class='green'>Ta</span></td>";
                  } else if (number % 5 == 0){
                    td += "<span class='purple'>Da</span></td>";
                  } else {
                    td += `${number}</td>`;
                  }
                  filtered += td;
                }
                return template;
              }

              // Display Results
              function displayResults(numbers, end){
                document.querySelector("#results").innerHTML = "";
                let result = filterValues(numbers, end);
                for (let i = 0; i < result.length; i++){
                  document.querySelector("#results").innerHTML += `<tr>${result[i]}</t>>`;
                }
              }
Summary

I use functions to encapsulate each section. Our program begins by listening for the click event on button then calling getValues(). We first grab the start & end values entered by user and store to a variable. To ensure values are type Integer(int) we use the parseInt() to convert any possible value supplied to a int or NaN.

We then pass our start & end variables as arguments through to our generateNumbers() function. We then store the returned array of numbers in a variable and pass as a argument to the displayResults() function.

Function explanation

generateNumbers() - Takes in startInt & endInt as arguments and uses a for-loop to dynamically push current Index to numbers array. Then returns numbers variable containing array of numbers.

filterValues() - This function takes in an array of numbers and the end int. I then declare 3 variables to serve as temporary templates for creating parts for HTML table.

First I check if the current iteration is equal to 5 so that I can display end the HTML Row and it to the DOM. I then use a For-loop to iterate over the numbers while checking for the following conditions:

  • If number is a multiple of 3 & 5 print "Tada"
  • If number is a multiple of 3 print "Ta"
  • If number is a multiple of 5 print "Da"

After the number has been run through the If statement it's then added to the filtered variable which will be added to the DOM. After all the checks have been performed an array of 5 table-data sets are returned.

displayResults() - Takes a array of number as arguments. Then a HTML table row is created as a string and assigned to a variable to serve as a template. A For-loop is then used to iterate over each number and insert into the template variable using string interpolation.
